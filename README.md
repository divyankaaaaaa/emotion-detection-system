# 🎭 Emotion Detection System

A real-time emotion detection web application that uses a webcam to analyze facial expressions and detect emotions like happiness, sadness, anger, and more.

---

## 🚀 Features

- 🎥 Real-time webcam feed in browser
- 😊 Emotion detection using DeepFace
- 🧠 Face detection using OpenCV
- 🌐 Flask-based web application
- ⚡ Live emotion labels with confidence score

---

## 🛠️ Tech Stack

- Python
- OpenCV
- DeepFace
- Flask

## ▶️ How to Run

### 1. Clone the repository

git clone https://github.com/divyankaaaaaa/emotion-detection-system.git

cd emotion-detection-system


### 2. Install dependencies

pip install -r requirements.txt


### 3. Run the app

python app.py


### 4. Open in browser

http://127.0.0.1:5000/
